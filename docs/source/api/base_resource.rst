Base API Resource
=================

All resource objects you will be using in Prevision.io's Python SDK inherit
from this base parent class.

In the SDK, a resource is an object that can be fetched from the platform,
used in your code, updated, deleted... :class:`previsionio.experiment.BaseExperiment`,
:class:`previsionio.dataset.Dataset` and :class:`previsionio.model.Model` are
all resources.
